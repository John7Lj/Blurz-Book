## **Critical Errors:**

### 1. **Missing `.startup` or proper initialization**
```assembly

.startup         ; Add this line - it initializes DS register
```

### 2. **Missing stack size**
```assembly
.stack 100h      ; Must specify size, not just .stack
```

### 3. **Wrong register for INT 21h output**
```assembly
; WRONG:

mov al, array[bx]

; CORRECT:
mov ah, 02h
mov dl, array[bx]    ; Use DL, not AL!
int 21h
```
